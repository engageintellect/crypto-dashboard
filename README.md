# crypto-dashboard

## Description:

When to buy, sell, and hold crypto? This dashboard will help you make the best decisions.

## How to contribute:

1. Fork the repo
2. Make your changes
3. Submit a pull request

## How to use:

1. Clone the repo
2. Run `pnpm install`
3. Run `pnpm start`
4. Go to `localhost:5173` in your browser
